# Tahra Morsi — Clinical Future Portfolio

This is a complete static GitHub Pages website. No Node.js, LaTeX, or package installation is required.

## Add Tahra's photo

1. Choose a clear professional portrait.
2. Rename it to tahra-photo.jpg.
3. Place it inside the assets folder.

Required path:

~~~text
assets/tahra-photo.jpg
~~~

The TM artwork remains visible until the photo is added.

## Test on Ubuntu

Open a terminal inside this folder and run:

~~~bash
python3 -m http.server 8000
~~~

Then open http://localhost:8000 in a browser. Press Ctrl+C to stop the local server.

## Replace the current GitHub Pages site

1. Open the Tahra124/tahra.github.io repository.
2. Delete or replace the old index.html, styles.css, script.js, README.md, and assets folder.
3. Upload every file and folder from this package to the repository root.
4. Commit the changes.
5. Wait a few minutes, then refresh https://tahra124.github.io/tahra.github.io/.

GitHub Pages is already configured, so it should rebuild automatically after the new files are committed.
